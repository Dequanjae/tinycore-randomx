# tinycore-randomx

A hyper-lightweight Monero (XMR) RandomX mining engine and live dashboard built with Rust for Tiny Core Linux.

### 📋 What You Need Before Starting
* **A Monero Wallet Address:** You will need a wallet address to receive your mining payouts. If you don't have one, download **Cake Wallet** on your phone or desktop and generate a Monero (XMR) address.

### 🚀 How to Run It
Open your Tiny Core Linux terminal and copy-paste this block (or run these 3 simple commands) to download, permission, and run the installer:

```bash
wget https://raw.githubusercontent.com/Dequanjae/tinycore-randomx/main/install.sh
chmod +x install.sh
./install.sh
